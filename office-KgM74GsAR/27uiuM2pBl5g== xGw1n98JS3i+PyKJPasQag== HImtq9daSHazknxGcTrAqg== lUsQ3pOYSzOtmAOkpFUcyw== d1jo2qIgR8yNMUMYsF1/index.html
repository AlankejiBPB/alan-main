* 预先准备

确保电脑拥有足够的C盘空间

提示：可以用电脑浏览器再开一个此网页，一个用来浏览教程，一个用来打开网站。

* 第一步 - 下载工具

下载 Office 软件部署工具：点击跳转下载

指南：

进入页面 ->

点击 下载 ->

打开 下载的程序 ->

勾选 左下角的对勾 ->

点 Continue ->

点击 桌面 ->

点击 新建文件夹 ->

输入 office ->

点击 确定 ->

再次点击 确定 ->

双击打开 桌面上的office文件夹

* 第二步 - 自定义软件（本品亮点）

进入 Office 版本自定义工具：点击进入

指南：

选择系统种类(一般都是 64位 ，不知道的百度查怎么看系统位数) ->

点击 Office 套件 并选择 Office LTSC Professional Plus 2024 - Volume License ->

按需选择要不要安装下面三种软件，如果需要均选择 LTSC Professional 版本 ->

来到 应用 ，按需选择要安装的软件 ->

点 下一个 ->

主要语言 选择 简体中文(中国) ->

点击 下一个 ->

打开 关闭正在运行的应用程序 ->

点击 下一个 ->

再次点击 下一个 ->

再次点击 下一个 ->

叕点击 下一个 ->

鼠标往右挪到空白地方，滚轮往下滚，点击 完成 ->

点击 右上角 导出 ->

弹出窗口 选择第一项 ->

点击 确定 ->

勾选 我接受许可协议中的条款 ->

文件名 那栏删除原来的填写 config ->

点击 导出 ->

如果提示有风险点击 保留 ->

将下载的文件 移动到 刚才打开的 office 文件夹

* 第三步 -下载安装 Office

在office文件夹内，点击 路径栏 （百度：Windows资源管理器路径栏是哪个），注意点空白部分，不要点有字的部分 ->

输入 cmd ->

复制以下命令，右键粘贴到弹出的黑窗口：

setup /download config.xml

按下回车，如果弹出窗口点击“是”，耐心等待命令执行，执行过程不会有输出，请勿关闭这个窗口，可以挂后台 ->

等到它出现以 > 结尾的一行并且光标闪烁时，复制以下命令，右键粘贴到黑窗口：

setup /configure config.xml

按下回车，如果弹出窗口点击“是”，此时 Office 开始自动安装你选择的软件，耐心等待安装完成

* 第四步 - 激活软件

点击下载 HEU_KMS

重要提示：请先提前关闭杀毒软件，否则会误报，原因是软件没有数字签名。

软件本身绿色安全，并且已在 Github 上开源（开源地址），请放心使用，保证绿色安全无后面！感谢信任！

有疑问可自行百度验证

打开软件 ->

点击 是 ，等待加载->

只打开 Office TSForge、Office Hook、Office KMS 三项（其余关闭）->

点击 开始 ->

点击 确定 ->

关闭软件 ->

打开杀毒软件 ->

在开始菜单就可以看到安装的 Office 了

* 更新软件

打开安装的任意一款软件 ->

点击右上角 文件 ->

点击 账户 ->

点击右侧 更新选项 ->

点击 立即更新
